## Sprint 1 deliverable (submit as Interview Practice App)

### Must-have requirements
- Single-page UI (Streamlit or Gradio)
- OpenAI API call using one of the allowed models
- 5+ system prompt variants using different techniques
- Tune at least one model parameter (temperature or top_p)
- Add at least one security guard:
  - input validation
  - prompt injection defense
  - refusal template

### Definition of done (Sprint 1)
- The app runs locally
- A user can submit an input and get a structured response
- Prompt variant selection works
- Safety guard blocks at least one invalid/malicious pattern
- Basic tests pass
